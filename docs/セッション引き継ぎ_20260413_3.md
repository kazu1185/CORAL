## 領収書の要件まとめ（議論結果）

- **PDF生成 → ダウンロード**（将来は自動印刷も予定）
- **予約詳細画面からいつでも発行可能**（全ステータスで有効）
- **CO時の自動発行は不要**（ユーザー指示で削除済み）
- **分割発行**: 明細行を選択して部分的な領収書を発行可能
- **宛名選択UI**: ゲスト名 / 法人名（company_name登録時のみ） / 直接入力の3択
  - 直接入力 → `guests.receipt_addressee` に保存（次回以降のデフォルト値）
- **但し書き**: デフォルト「宿泊代として」、編集可能
- **内訳**: 日別明細
- **税表示**: 税率ごとに分けて表示（10%対象額、消費税、宿泊税）— インボイス制度対応
- **グループ予約**: 各室ごと個別 or グループ全体1枚を発行時に選択可能
- **収入印紙**: 5万円以上の場合に「収入印紙貼付欄」の枠線を印字
- **ロゴ**: モノクロロゴをPDFヘッダーに配置
- **採番**: `R-YYYY-NNNN`（年次連番）
- **再発行**: 宛名・但し書き変更可能、赤字で「再発行（N回目）」表示

## API仕様

### 領収書発行 — POST /api/v1/documents/receipt
```json
{
  "reservation_id": 123,
  "addressee": "山田太郎",
  "description": "宿泊代として",
  "payment_method_id": 1,
  "charge_ids": [10, 11, 12],
  "group": false,
  "save_addressee": true
}
```
レスポンス: `{ "document_id": 1, "message": "領収書を発行しました" }`

### PDF取得 — GET /api/v1/documents/:id?format=pdf&download=1
- `format=pdf`: PDFバイナリ返却
- `download=1`: Content-Disposition: attachment

### 帳票一覧 — GET /api/v1/documents?reservation_id=123
- 予約に紐づく発行済み帳票一覧

### 再発行 — POST /api/v1/documents/:id/reissue
```json
{
  "addressee": "株式会社サンプル",
  "description": "宿泊代として"
}
```

### 予約詳細API — GET /api/v1/reservations/:id
- レスポンスに `documents` 配列（発行済み帳票一覧）と `receipt_addressee`（ゲスト保存宛名）を追加済み
