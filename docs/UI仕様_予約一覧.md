# 予約一覧 UI仕様書

UIモック（pms_reservation_list.html）を参考にした画面仕様。

---

## 1. 画面の役割

予約の検索・一覧表示・詳細確認を行う中核画面。ダッシュボードのCI/COリスト「全件表示」や各種フィルタ遷移の着地先。

---

## 2. レイアウト構成

```
┌─ サイドナビ ─┬──────────────────────────────────────────────────┐
│              │ ヘッダー（ダッシュボードと共通）                       │
│ ☰ 予約一覧   ├──────────────────────────────────────────────────┤
│   ← active   │ ツールバー（検索 + フィルタ + CSV + 手動予約）         │
│              │ フィルタパネル（展開/折りたたみ）                      │
│              │ ステータス別件数チップ                                │
│              │ テーブル                                            │
│              │ ページネーション                                     │
│              ├──────────────────────────────────────────────────┤
│              │         ← 詳細ドロワー（右からスライドイン）           │
└──────────────┴──────────────────────────────────────────────────┘
```

---

## 3. ツールバー

### 3-1. 検索ボックス

- プレースホルダー：「ゲスト名・予約番号で検索」
- フォーカス時にブルーボーダー＋シャドウ
- 左にアイコン 🔍

### 3-2. フィルタボタン

- 「フィルタ ▼」ボタン。押下でフィルタパネルの展開/折りたたみ
- アクティブ時はブルー背景

### 3-3. 右側ボタン

- 「📥 CSV出力」（セカンダリボタン）
- 「＋ 手動予約入力」（プライマリボタン・青背景）

---

## 4. フィルタパネル

ツールバーの「フィルタ」ボタンで展開される折りたたみパネル。

### 4-1. フィルタ項目

| グループ | 選択肢（チップ型・複数選択可） |
|---------|--------------------------|
| OTA | 🔴じゃらん / 🟤楽天 / 🔵Booking / 🟢直販 / 🟠電話 / 🟡Expedia / 🟣Agoda |
| 室タイプ | シングル / ダブル / ツイン / デラックス / スイート |
| CI日 | 日付範囲指定（FROM〜TO） |

- チップは選択時にブルー背景
- 全フィルタはAND検索で組み合わせ可能

---

テーブル上部に横並び表示。フィルタチップとしても機能する。

## 6. テーブル

### 6-1. 列構成

| 列 | 表示内容 | ソート |
|----|---------|-------|
| CI日 | 日付 + 泊数バッジ（例: 4/10 `2泊`） | ✅ デフォルト降順 |
| CO日 | 日付 | ✅ |
| ゲスト名 | 名前行 + ローマ字行（2段） + バッジ群 | ✅ |
| OTA | カラードット + OTA名 | ✅ |
| 予約番号 | モノスペースフォント（例: BK-44521） | - |
| 室タイプ / 部屋 | 部屋番号 + タイプ、未アサインは黄色テキスト | - |
| ステータス | ステータスバッジ（色付きピル型） | ✅ |
| 金額 | 右寄せ・等幅数字（例: ¥28,000） | ✅ |

### 6-2. ゲスト名列のバッジ（名前の右に表示、flex-wrap）

| バッジ | 表示条件 | 色 |
|-------|---------|-----|
| ⭐VIP | is_vip = true | ピンク背景 #FDF2F8 |
| 🔁×4 | visit_count >= 2 | 琥珀背景 #FEF3C7 |
| 🔗連結 | guest_linksに紐付きあり | 紫背景 #F5F3FF |
| ⚠穴 | 連結予約にgapあり | 赤背景 #FEF2F2 |

表示順: VIP → リピーター → 連結 → 穴

### 6-3. ステータスバッジの色

| ステータス | 背景 | テキスト | 補足 |
|-----------|------|---------|------|
| 確定 | #DBEAFE | #2563EB | |
| CI済 | #DCFCE7 | #16A34A | |
| CO済 | #EDE9FE | #7C3AED | |
| キャンセル | #F1F5F9 | #94A3B8 | 打消線付き |
| ノーショー | #FEE2E2 | #DC2626 | |

### 6-4. 長い名前の対応

- ゲスト名セル: `min-width: 160px; max-width: 260px; white-space: normal`
- 名前とバッジは`flex-wrap: wrap`で自然に折り返し
- ローマ字行は`text-overflow: ellipsis`で省略

### 6-5. 行の振る舞い

- ホバー: 薄いグレー背景
- クリック: 詳細ドロワーが開く + 行がブルー背景（選択状態）

---

## 7. ページネーション

- 15件/ページ
- 表示: 「1–15 / 42件」
- ページボタン: ◀ [1] [2] [3] ▶
- アクティブページはブルー背景

---

## 8. 詳細ドロワー

行クリックで右からスライドインする詳細パネル。

### 8-1. 構成

```
┌─ ドロワーヘッダー ──────────────────┐
│ ゲスト名  部屋番号            [✕]   │
├─────────────────────────────────────┤
│                                     │
│ ■ 予約情報                           │
│   ステータス・OTA・予約番号            │
│   CI日・CO日・室タイプ・部屋番号       │
│                                     │
│ ■ ゲスト情報                         │
│   氏名（VIPバッジ）・ローマ字・人数    │
│   宿泊回数                           │
│   [🔁 宿泊履歴を表示 ▼] ← クリック展開│
│     過去の宿泊実績（アコーディオン）    │
│                                     │
│ ■ 連結予約タイムライン（該当時のみ）    │
│   OTAカラーセグメントバー             │
│   穴がある場合は赤破線セグメント        │
│                                     │
│ ■ 売上情報                           │
│   1泊1行のテーブル                    │
│   小計・消費税・合計                   │
│                                     │
├─────────────────────────────────────┤
│ [予約詳細画面へ] [アサインボード]       │
│ [予約と連結] [領収書発行]              │
└─────────────────────────────────────┘
```

### 8-2. 宿泊履歴アコーディオン

- リピーター（visit_count >= 2）の場合のみ表示
- 「🔁 宿泊履歴を表示 ▼」ボタンをクリックで展開
- 各履歴行: 日付 / 部屋・タイプ / OTAドット+名前 / 金額

### 8-3. 連結予約タイムライン

- 各予約がOTAカラーのセグメントとして横バー表示
- 穴がある場合は赤破線のセグメント
- 下にOTA凡例

### 8-4. ドロワーの振る舞い

- 背景にオーバーレイ（半透明黒）
- オーバーレイクリックで閉じる
- スライドイン アニメーション（右→左、0.25秒）
